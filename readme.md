## 🎓 Campus Issue Management Platform

**AI-powered platform for reporting, prioritizing, and resolving campus issues.**

Students can report issues with images and location details while administrators manage them through a centralized dashboard featuring analytics, role-based access control, AI-assisted priority scoring, duplicate detection, and real-time notifications.

**Highlights**

- 🔐 JWT Authentication & Role-Based Access Control
- 🤖 AI Priority Scoring & Duplicate Detection
- 📸 Image Uploads & Location Tagging
- 📊 Admin Dashboard & Analytics
- 🔔 Real-Time Notifications
- 🎨 Modern Glassmorphism UI

**Tech Stack**

`React` `Node.js` `Express.js` `PostgreSQL` `JWT` `Cloud Storage`

**🔗 Links**

- 🌐 Live Demo: https://campus-issue-frontend-five.vercel.app/

## 🚆 GroupRail

**AI-powered collaborative train journey planner for groups.**

A real-time travel planning platform where users can create groups, invite friends, chat instantly, manage itineraries, receive AI boarding recommendations, and organize trips together.

**Highlights**

- 🔐 JWT Authentication
- 👥 Invite Codes
- 💬 Real-Time Chat (Socket.IO)
- 🤖 AI Boarding Recommendations
- 🪑 Seat Preference Management
- ⚡ Zustand State Management

**Tech Stack**

`React` `Node.js` `Express.js` `MongoDB` `Socket.IO` `Tailwind CSS`
